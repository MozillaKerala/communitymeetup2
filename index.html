Frequently Asked Questions (FAQ)

  • Q1) Is this a technical event?
    Ans: No

    Q2) Can anyone attend the event?
    Ans: No. The event is invite only.

    Q3) How many seats are there?
    Ans: 20.

    Q4) I don't have technical proficiency in HTML or C++ or anything, but I'm passionate about Mozilla. Can I attend?
    Ans: Yes!

    Q5) Who is organizing/sponsoring this event?
    Ans: This is a Mozilla event, which means Mozilla is organizing this event through its representatives.

    Q6) What is the event date?
    Ans: 19th January 2014

    7) Event timing?
    Ans: 9.30 AM to 5.00 PM

  • Q8) What is the event agenda?
    Ans: The event agenda is still being planned. A tentative agenda is given below under the heading Agenda.

    Q9) Who can attend this event?
    Ans: This meetup is ONLY for those who wish to be active in our community. Please read the NOTE given below and decide if this is fit for you. If in doubt, mail to events@mozillaindia.org

    NOTE: This event is invite only and is aimed at laying the foundation of Mozilla community in Kerala. So this meetup is aimed at those who love Mozilla and wish to be active volunteers in our who can take up responsibilities. We all have other important things in life, such as school, college, work etc. So we are not expecting anyone to put off any such other things and work for the community. Its more like, you have to have a strong passion for Mozilla and the values for which we stand for and should be willing to give it your best possible.

    So if you are not a hard core Mozilla fan (or a wannabe hard core Mozillian), you might not want to attend this meeting. You can still participate in other public Mozilla events we organize, including but not limited to the upcoming developer events.
